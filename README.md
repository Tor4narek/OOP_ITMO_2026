# Документация курса Объектно-ориентированное программирование

Добро пожаловать в этот репозиторий! Здесь собраны материалы и условия лабораторных работ по курсу ООП. Репозиторий будет регулярно обновляться по мере прохождения курса.

## Лабораторные работы

**[Лабораторная работа №1](/Lab1.md)**  
    **Цель:** освоение базовых принципов объектно-ориентированного программирования в C# и навыков проектирования консольного приложения.

**[Лабораторная работа №2](/Lab2.md)**  
    **Цель:** освоение работы с обобщениями (Generics), делегатами и лямбда-выражениями в C#, а также использование LINQ для обработки данных из консольного приложения. Развитие навыков создания расширяемой архитектуры для работы с коллекциями и анализа данных.

**[Лабораторная работа №3](/Lab3.md)**  
    **Цель:** освоение асинхронного программирования (async/await), работы с файловой системой и JSON, а также создание отдельной библиотеки для сериализации и десериализации.

**[Лабораторная работа №3](/Lab4.md)**  
    **Цель:** освоение принципов работы с реляционными базами данных и взаимодействия с ними из C#-приложения с использованием **Entity Framework Core**.
